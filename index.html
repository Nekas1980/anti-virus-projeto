// 01 — FUNCIONALIDADES

O que faz
o scanner

F-001

Detecção SHA-256

Calcula a impressão digital única de cada ficheiro e compara com a base de assinaturas de malware.

F-002

Scan Recursivo

Analisa pastas e todas as subpastas em profundidade com um único comando.

F-003

Quarentena Automática

Move ficheiros infectados para uma pasta isolada sem apagar, preservando evidências.

F-004

Relatório JSON

Gera relatório completo com status, hash SHA-256 e nome da ameaça de cada ficheiro.

F-005

Zero Dependências

Usa apenas bibliotecas nativas do Python. Sem instalação de pacotes externos.

F-006

Open Source

Código aberto e livre para usar, modificar e contribuir. Licença MIT.

// 02 — INSTALAÇÃO

Como usar

01

Pré-requisitos

Precisas de Python 3.8+ instalado. Sem bibliotecas externas necessárias.

02

Clonar o repositório

git clone https://github.com/Nekas1980/anti-virus-projeto.git

03

Adicionar assinaturas

Coloca o ficheiro assinaturas.json com os hashes de malware na pasta do projeto.

04

Correr o scanner

python Projeto_virus.py — indica o diretório e o scanner faz o resto.
